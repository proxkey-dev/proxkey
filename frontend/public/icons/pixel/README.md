# Pixel Icons from The Noun Project

This directory contains pixel-style icons downloaded from [The Noun Project](https://thenounproject.com/browse/icons/term/pixel/).

## How to Add Icons

1. Visit [The Noun Project - Pixel Icons](https://thenounproject.com/browse/icons/term/pixel/)
2. Search for the icon you need
3. Download the SVG file
4. Save it in this directory with a descriptive name (e.g., `pixel-arrow.svg`, `pixel-star.svg`)
5. Use it in your components via the `PixelIcon` component:

```tsx
import { PixelIcon } from '@/components/ui/PixelIcons';

// Basic usage
<PixelIcon name="pixel-arrow" className="w-6 h-6" />

// With custom size and color
<PixelIcon name="pixel-star" size={32} color="#ff6a3d" />
```

## Available Icons

Add icons here as you download them from The Noun Project. Common icons to consider:
- `pixel-arrow.svg` - Arrow icons
- `pixel-star.svg` - Star icons  
- `pixel-heart.svg` - Heart icons
- `pixel-art.svg` - Art/graphic icons
- `pixel-icon.svg` - Generic pixel icon
