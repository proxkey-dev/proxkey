# Security

If you believe you have found a security vulnerability in ProxKey, please **do not** open a public issue.

Report it privately using **GitHub → Security → Report a vulnerability** on the affected repository under [proxkey-dev](https://github.com/proxkey-dev), or another channel your maintainers publish for production deployments.

Include:

- Description of the issue and impact
- Steps to reproduce (proof-of-concept if possible)
- Affected components or paths (e.g. `frontend/server`, `apps/api`)

We aim to acknowledge reports promptly and coordinate disclosure after a fix is available.
